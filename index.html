Shortilicious

A nice looking URL Shortening site!

Very easy install with a nice GUI all you need to do is upload the files to your site. Then visit your site address it will then prompt you to go to the installation wizard from there everything will be done for you. There is no need to edit a config.php file or upload a sql file to PHPAdmin.
